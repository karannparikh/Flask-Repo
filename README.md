# Flask-Repo
# Project Title - A minimal Flask application.

## Setup and Installation
Create an environment

    $ mkdir myproject
    $ cd myproject
    $ python3 -m venv .venv
Activate the environment

    $ . .venv/bin/activate
Within the activated environment, use the following command to install Flask:

    $ pip install Flask

## Installation

Within the activated environment, use the following command to install Flask:

    $ pip install Flask
